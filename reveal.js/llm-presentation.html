LLMs as Next-Token Predictors

Large Language Models work by predicting the next token in a sequence

  • Take input text
  • Predict most likely next token
  • Add token to sequence
  • Repeat process

Tokenization

Before processing, text must be split into tokens

  • Tokens are the basic units LLMs work with
  • Can be words, subwords, or characters
  • GPT-2 uses Byte Pair Encoding (BPE)
